# TradeAlert v1 UI Direction

The phone UI should feel like a working tradesperson's control surface: calm, quick to scan, and not cluttered.

Reference mockup: `tradealert-v1-phone-ui-mockup.png`

Implemented direction:

- Warm off-white app background.
- Charcoal text with trade-green action/status accents.
- Separate sections for phone agent, jobs, invoices/photos, van stock, customers, Maps, and Screwfix.
- Minimal card depth, no nested cards.
- Large touch targets and short labels for field use.
